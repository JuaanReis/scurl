Referência da API

Base URL para todas as requisições: https://scurl.vercel.app

Autenticação & Limites

A API é pública e não requer tokens de acesso. O controle é feito via Rate Limiting por endereço IP.

Recurso Configuração / Resposta
Rate Limit Configurável via config["rate_limit"]. Retorna 429 Too Many Requests se excedido.
Concorrência Máximo de 20 scans simultâneos. Retorna 503 Service Unavailable se o limite for atingido.

Endpoints Disponíveis

POST /analyze

Executa a engine de análise heurística completa sobre uma URL específica.

Request Body (JSON)

CampoTipoObrigatórioDescrição
urlHttpUrlSimURL alvo validada.
use_cacheboolNãoSe true, retorna dados do banco SQLite se existirem. (Default: false)
POST /analyze
Content-Type: application/json

{
  "url": "https://example.com",
  "use_cache": false
}

GET /scans

Retorna lista de análises recentes (requer cache ativo no servidor).

ParâmetroTipoPadrãoLimites
limitint501 a 200
offsetint0>= 0

GET /scans/{identifier}

Busca um resultado completo usando o scan_id (UUID) ou o url_hash (SHA-256).

Schemas de Resposta

ScanResponse

Objeto principal retornado após uma análise bem-sucedida.

{
  "status": "ok",
  "result": {
    "score": 7.8,
    "risk_level": "very low",
    "verdict": "safe"
  },
  "stats": {
    "rules_total": 36,
    "rules_triggered": 3
  }
}

Escala de Risco (result.risk_level)

ScoreNível de Risco
0 - 20very low

TargetResponse (Dados Coletados)

Abaixo, os campos detalhados das seções do alvo:

1. Identity (URL & Domínio)

CampoDescrição
registered_domainDomínio principal (ex: google.com).
is_homographIndica se há uso de caracteres internacionais suspeitos.
domain_entropyCálculo de Shannon para detectar domínios gerados aleatoriamente.

2. Network & DNS

CampoDescrição
asnNúmero do Sistema Autônomo e Organização.
timingsBreakdown de latência (DNS, TCP, TLS, TTFB) em milissegundos.
spf / dmarcPresença de registros de proteção de e-mail.

3. Content (Análise HTML)

CampoDescrição
scripts / iframesContagem de elementos externos e inline.
formsDetecção de campos de senha e destinos de formulários.
spaVerifica se o site é uma Single Page Application.
